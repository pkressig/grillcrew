# GrillCrew FCTC

Mobile-first Helfer- und Einsatzplanung für den FC Thusis-Cazis.

## Aktueller Stand

Dieses Repository enthält die verbindliche Produktgrundlage für Version 1. Es dient als gemeinsame Arbeitsbasis für Pascal, ChatGPT und Claude Code.

Noch nicht implementiert:
- Anwendungscode
- Datenbank
- Authentifizierung
- Hosting

## Ziele von Version 1

- Saisons, Events und Schichten einfach verwalten
- Helfer ohne komplizierte Anmeldung eintragen lassen
- optionale Helferkonten mit Datenübernahme ermöglichen
- Familien, Kinder und Sollstunden korrekt abbilden
- effektive Arbeitszeiten und Vergütungsart erfassen
- offene Schichten, Auszahlungen und Saisonstände übersichtlich zeigen
- bestehende Excel-Daten importieren
- Excel und manuelle Parallelführung langfristig ersetzen

Der Chat ist nicht die dauerhafte Wahrheit. Die Dokumentation in diesem Repository ist die Single Source of Truth. Neue Entscheidungen werden zuerst dokumentiert und danach umgesetzt.
