Build GrillCrew into a secure, polished, mobile-first SaaS for volunteer event staffing while preserving
strict tenant isolation, approachable workflows, explicit human product authority, and auditable delivery.

The Git repository is durable project memory. Paperclip coordinates goals, task contracts, execution,
reviews, approvals, costs, and audit history. The Product Owner remains the final authority.
